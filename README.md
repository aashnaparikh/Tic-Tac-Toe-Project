# Tic Tac Toe  

A classic Tic Tac Toe game implemented using JavaScript, HTML, and CSS. Play against a friend or challenge yourself with a basic AI opponent.  

## Features  

- Two-player mode for local play.  
- Interactive and responsive user interface.  
- Highlighting of the winning line.  
- Option to restart the game without refreshing the page.  

## Technologies Used  

- JavaScript for game logic.  
- HTML for structure and layout.  
- CSS for styling and animations.  

## Future Improvements  

- Add an AI mode with varying difficulty levels.  
- Include sound effects and animations for better user experience.  
- Track player scores across multiple rounds.
